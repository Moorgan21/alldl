# alldl

ربات تلگرامی که لینک از اینستاگرام، تیک‌تاک و پینترست می‌گیره و فایل (ویدیو/عکس) رو دانلود و برای کاربر ارسال می‌کنه.

## نصب

```bash
pip install -r requirements.txt
```

`ffmpeg` هم باید نصب باشه (برای merge کردن ویدیو/صدا و ساخت thumbnail).

## تنظیم

1. `.env.example` رو کپی کن به `.env` و توکن بات تلگرامت رو بذار توش:
   ```
   BOT_TOKEN=xxxxx
   ```
2. (اختیاری، برای دانلود از اینستاگرام) کوکی حساب اینستاگرام رو با اکستنشنی مثل Cookie-Editor به‌صورت JSON export کن و بذار تو `cookies/instagram.json` — ربات موقع استارت خودش تبدیلش می‌کنه به فرمت مورد نیاز.

## اجرا

```bash
python3 bot.py
```

## قابلیت‌ها

- دانلود پست/ریلز/استوری اینستاگرام
- دانلود ویدیو/عکس تیک‌تاک
- دانلود ویدیو/عکس پینترست
- استخراج duration/thumbnail واقعی ویدیو برای نمایش درست تو تلگرام
